# Logic-Design
This project implements a majority voter for five inputs using an Arduino Uno. It demonstrates a complete digital logic design flow—from truth table and Karnaugh map simplification to hardware implementation. The system uses two LEDs to visually indicate the result:

Red LED lights up when 3 or more inputs are HIGH (majority).

Green LED lights up when fewer than 3 inputs are HIGH (minority).

This project is ideal for learning combinational logic, Boolean algebra, and microcontroller basics.
